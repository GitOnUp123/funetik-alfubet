# Fünetik Alfübet: “Keep It Simple” Phonetic Alphabet

## Overview

The **Fünetik Alfübet** is a simplified, learner‑focused phonetic alphabet designed to make English pronunciation clearer, more intuitive, and far easier for ESL learners than the International Phonetic Alphabet (IPA). It follows a **One Sound, One Character** principle, using familiar Latin characters and minimal diacritics to maximize readability.

This draft is released openly to invite community review from linguists, ESL educators, and learners.

\---

## Motivation

English spelling is irregular, and IPA can overwhelm beginners. The Fünetik Alfübet aims to balance **phonetic accuracy** with **pedagogical simplicity**, making pronunciation teaching more effective.

\---

## Key Features

* **One Sound, One Character**
* **Minimal and intuitive diacritics** (ā, ē, ī, ō, ò, ŏ)
* **Schwa omitted intentionally** in favour of underlying vowels (u, ʊ, i)
* **Clear rhotic vowels** using ʊr or vowel+r
* **No diphthong symbols** — natural transitions represented simply
* **Dialect transparency** — based on Canadian English, adaptable to others
